-----Instalaciones previas para el uso de la aplicación-----

Se debe instalar la libreria redis de python: pip install redis
Se debe instalar la libreria colorama de python: pip install colorama
Se debe instalar la libreria art de python: pip install art
Se debe instalar la libreria pyfiglet de python: pip install pyfiglet
Se debe instalar la libreria PySide2 de python: pip install  PySide2
