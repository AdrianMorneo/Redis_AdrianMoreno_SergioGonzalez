-----SOBRE LA APLICACIÓN-----

Esta aplicación gestiona un santuario de animales, donde se han desarrollado 2 Crud, uno de animales y otro de
colaboradores.

Los animales se registran con sus respectivos atributos y los colaboradores igual.
Se han utilizado diccionario para colaboradores y Strings para animales(dos formas diferentes para hacer lo mismo).

Los animales se pueden apadrinar y desapadrinar siempre que exista un animal y un colaborador.

El proyecto se ha apoyado en la ayuda de ChatGPT sobre t0do para el código de la interfaz gráfica, hemos aprendido a
adaptarlo a nuestras necesidades, no ha sido realmenteun copiar y pegar.

Se ha utilizado funciones lambda en modificar y agregar, tanto animales como colaboradores, estando en contra de esta
praxis, ya que no la controlamos, pero no se consiguió que la aplicación se ejecutara de la manera que queríamos sin el
uso de este recurso.

Esperamos que la aplicación sea de su agrado y disfrute de los extravatantes colores y bonitas fuentes en la interfaz de
consola y de la colosal interfaz gráfica, t0do creado con una exaustiva investigación de diferentes librerías a usar.



-----ANTES DE EMPEZAR-----

Para el desarrollo y la ejecución correcta del programa, se recomienda usar PyCharm, con el interprete Python 3.8.

El inicio de la ejecución, da una elección en consola, a la que da la opción al usuario para hacer uso de interfaz
gráfica o uso en consola.

Ambas opciones están trabajadas y optimizadas para el uso.

La opción de Interfaz Gráfica, tiene desabilitada la opción de apadrinar y desapadrinar. La opción de consola sí tiene
esta opción.

Instale las librerías que a continuación se detallan.


-----Instalaciones previas para el uso de la aplicación-----

Se debe instalar la libreria redis de python: pip install redis
Se debe instalar la libreria colorama de python: pip install colorama
Se debe instalar la libreria art de python: pip install art
Se debe instalar la libreria pyfiglet de python: pip install pyfiglet
Se debe instalar la libreria PySide2 de python: pip install  PySide2
